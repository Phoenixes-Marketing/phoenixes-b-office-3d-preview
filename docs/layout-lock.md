## 鎖定規則

目前版本為「純空間格局確認版」。除非使用者明確要求調整整體格局、牆面位置、牆面尺寸、柱位、門洞、矮牆或地坪高程，後續修改只能調整：

- 家具
- 櫃體
- 燈光
- 材質
- 裝飾
- 植栽
- 衛浴設備
- 局部軟裝

不得任意改動下列格局骨架：

- 牆面：46 段
- 門：15 組
- 柱：16 根
- 矮牆：4 段
- 門口階梯：3 段
- 室內地坪：+60cm
